# Installation

## Stable release

To install monte_carlo, run this command in your terminal:

```sh
uv add monte_carlo
```

Or if you prefer to use `pip`:

```sh
pip install monte_carlo
```

## From source

The source files for monte_carlo can be downloaded from the [Github repo](https://github.com/KarunyaShirali/monte_carlo).

You can either clone the public repository:

```sh
git clone https://github.com/KarunyaShirali/monte_carlo
```

Or download the [tarball](https://github.com/KarunyaShirali/monte_carlo/tarball/main):

```sh
curl -OJL https://github.com/KarunyaShirali/monte_carlo/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd monte_carlo
uv sync
```
